# Homebrew Tap for bawa
Homebrew tap for installing `bawa` on Linux (Ermgamba I don't have an Apple device I'm not rich)

[`bawa` is a tui game save organizer](https://github.com/sarowish/bawa)

## Installation
```bash
# Adding the tap
brew tap cometpuppy/homebrew-bawa

# Install bawa
brew install bawa
